Javascript

La categoria 'JavaScript Essentials' offre una panoramica fondamentale delle caratteristiche chiave del linguaggio JavaScript, dalle basi della sintassi e della struttura del codice alla gestione degli eventi e alle interazioni con il DOM (Document Object Model). Questa categoria è progettata per aiutare i principianti a ottenere una solida comprensione dei concetti fondamentali di JavaScript e per fornire ai programmatori più esperti un rinfresco sui concetti di base. Attraverso esempi pratici e spiegazioni dettagliate, gli studenti avranno l'opportunità di acquisire competenze essenziali per la creazione di applicazioni web dinamiche e interattive utilizzando JavaScript.

document

Il comando document in JavaScript è un oggetto che rappresenta l'intero documento HTML caricato nella finestra del browser. Questo oggetto fornisce metodi per manipolare il documento HTML e accedere ai suoi elementi.

Selezione degli elementi:

  1. getElementById(): Restituisce un riferimento all'elemento con l'ID specificato. 
    var element = document.getElementById("myId");
  2. getElementsByClassName(): Restituisce una collezione di tutti gli elementi con la classe specificata. 
    var elements = document.getElementsByClassName("myClass");
  3. getElementsByTagName(): Restituisce una collezione di tutti gli elementi con il nome del tag specificato. 
    var elements = document.getElementsByTagName("div");
  4. querySelector(): Restituisce il primo elemento che corrisponde al selettore CSS specificato. 
    var element = document.querySelector("#myId .myClass");
  5. querySelectorAll(): Restituisce una NodeList di tutti gli elementi che corrispondono al selettore CSS specificato. 
    var elements = document.querySelectorAll("div.myClass");

Manipolazione del contenuto:

  1. innerHTML: Proprietà che rappresenta il contenuto HTML di un elemento. 
    element.innerHTML = "<p>Nuovo contenuto</p>";
  2. textContent: Proprietà che rappresenta il testo di un elemento e dei suoi discendenti. 
    element.textContent = "Testo aggiornato";

Modifica degli stili:

  1. style: Proprietà che consente di modificare gli stili CSS di un elemento. 
    element.style.color = "red";

Gestione degli eventi:

  1. addEventListener(): Aggiunge un gestore di eventi a un elemento per un determinato tipo di evento. 
    element.addEventListener("click", function() {
  console.log("Elemento cliccato");
  });

Manipolazione della struttura del documento:

  1. createElement(): Crea un nuovo elemento HTML specificato dalla stringa. 
    var newElement = document.createElement("div");
  2. appendChild(): Aggiunge un nuovo nodo alla fine della lista dei figli di un elemento. 
    parentElement.appendChild(newElement);
  3. removeChild(): Rimuove un nodo figlio dall'albero DOM. 
    parentElement.removeChild(childElement);

window

In JavaScript, l'oggetto window rappresenta la finestra del browser. È l'oggetto globale e contiene variabili, funzioni e altri oggetti che sono disponibili globalmente all'interno del contesto di esecuzione JavaScript di una pagina web.

Gestione della finestra del browser:

  1. Apertura e chiusura della finestra:
    • window.open()
    • window.close()
    • window.closed
  2. Dimensionamento e posizionamento della finestra:
    • window.resizeTo()
    • window.moveTo()
  3. Scrolling:
    • window.scrollBy()
    • window.scrollTo()

Gestione della navigazione:

  1. URL e pagine:
    • window.location
    • window.location.href
    • window.location.reload()
    • window.location.replace()
  2. Storia della sessione:
    • window.history
    • window.history.back()
    • window.history.forward()
    • window.history.go()

Gestione degli eventi:

  1. Eventi della finestra:
    • window.onload
    • window.onresize
    • window.onbeforeunload

Timer e temporizzazione:

  1. Timer:
    • window.setTimeout()
    • window.setInterval()
    • window.clearTimeout()
    • window.clearInterval()
  2. Orologio:
    • window.Date()
